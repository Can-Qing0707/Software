# 基于大模型技术的软件实训教学结果检查评价与报表系统 — 开发计划

> 第十五届中国软件杯大赛 B1 赛题 | B 组（高职）| 出题企业：龙芯中科技术股份有限公司

---

## 一、项目概述

构建一套覆盖软件实训全流程的自动化、智能化检查评价体系，支持实训成果上传、智能核查、多维度评价、报表生成与导出、评价标准自定义等核心功能。

**硬性约束**：B/S 架构，必须部署于自主指令集 LoongArch 架构 + 银河麒麟高级服务器操作系统并运行通过（不满足视为 0 分）。

## 四、功能模块

### 模块 1：用户与权限管理
- 登录/注册/登出（JWT + bcrypt）
- 角色：管理员、教师、学生
- 权限控制：路由级（beforeEach）+ API 级（中间件）
- 个人信息管理

### 模块 2：课程与实训任务管理
- 课程 CRUD（管理员/教师）
- 实训任务 CRUD（关联课程）
- 任务包含：要求描述、评分标准、截止时间
- 学生任务列表与完成状态

### 模块 3：实训成果上传与解析
- 支持多格式上传：Word(.docx)、PDF、图片(png/jpg)、截图
- 前端预览：PDF 预览、图片预览
- 后端提取文件文本内容（docx/pdf 解析库）
- 调用 LLM 解析核心内容，提取关键信息
- 上传进度条、批量上传

### 模块 4：智能核查
- 提交内容 vs 任务要求 自动比对（LLM 驱动）
- 实训步骤完整性核查
- 逻辑漏洞初步识别
- 核查结果展示（通过/警告/不通过 + 详情）

### 模块 5：多维度评价管理
- 自定义评价指标（代码质量、文档规范性、功能实现度等）
- 指标权重分配（百分比，总和 100%）
- LLM 客观自动评分（一键触发）
- 教师主观评分入口（可覆盖/补充 LLM 评分）
- 最终得分加权计算
- 评分结果汇总展示

### 模块 6：报表生成与导出
- 个人实训评价报告（PDF 导出）
- 班级/课程统计报表（Excel 导出）
- 可视化图表：雷达图（多维能力）、柱状图（成绩分布）、饼图（状态统计）、折线图（趋势）
- 支持筛选条件（时间、课程、班级）

### 模块 7：系统管理
- LLM 服务配置（API 地址、Key、模型名），支持云端与本地大模型切换
- 评价指标模板管理
- 操作日志

### 模块 8（创新）：智能分析与建议（加分项）
- **薄弱点智能分析**：基于评价数据自动识别学生能力短板
- **个性化学习建议**：LLM 根据评分结果生成针对性改进建议
- **实训质量画像**：聚合全量数据，形成课程/班级/学生维度质量画像
- **趋势预警**：多轮实训成绩趋势分析，提前预警下滑风险

## 九、关键技术决策

| 决策项 | 选择 | 说明 |
|--------|------|------|
| LLM 方案 | 云端 API（兼容 OpenAI 协议） | 支持 OpenAI / 本地大模型双模式 |
| 文件解析 | docx 解析库 + pdf 解析库 + LLM | 先提取文本再送 LLM 分析 |
| 文件存储 | 本地文件系统 | 部署时映射至麒麟系统目录 |
| 认证方式 | JWT Token（72h 过期） | 无状态认证，适合 B/S 架构 |
| 部署架构 | B/S 架构 | 前后端分离，Nginx 反代 |
| 目标平台 | LoongArch + 银河麒麟 V10/V11 | 龙芯云平台远程虚机 |
| 报告生成 | gofpdf + excelize | Go 原生 PDF/Excel 库 |
